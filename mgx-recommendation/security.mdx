## Data Protection Layers

Spideo provides several layers of data protection:

1. **API Key Authentication**
Authenticate HTTPS requests using your [API Key](/docs/vocabulary/#API_KEY).

2. **API Gateway**
Secure and manage API traffic. Build in WAF to secure our services from malicious attacks

3. **IP Range Filtering**
Restrict access to trusted IP ranges.
