## Context awareness

Each content can describe a set of context where it is available.
Those context allow to be filtered out at request time, ensuring that each user will only see what they have access to.
Those filters can be passed directly at call time via the [availability filters DSL](/mgx-recommendation/availability-fitler/overview.mdx).

## Time awareness
